# Student Manager

A fullstack MERN application for managing students.

## Setup

### Prerequisites

- Node.js
- MongoDB

### Backend

1. cd server
2. npm install
3. Create a .env file with MONGO_URI=your_mongodb_connection_string
4. npm run dev

### Frontend

1. cd client
2. npm install
3. npm start

The app will run on http://localhost:3000
Backend on http://localhost:5000
